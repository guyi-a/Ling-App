> **日期**: 2026-04-02

## 🎯 优化目标

### 核心原则
- 🎨 **美观优先**: Material 3 卡片风格，圆角、阴影、渐变
- 💡 **体验流畅**: 动画过渡、加载反馈、错误处理
- 🧩 **功能完整**: 覆盖后端所有核心 API 功能
- 📱 **响应式**: 适配不同屏幕尺寸和暗黑模式

### 后端 API 对应界面需求

| 后端模块 | 对应界面功能 |
|----------|--------------|
| **认证 API** | 登录页、注册页、Token 管理 |
| **用户 API** | 用户资料页、设置页 |
| **会话 API** | 会话列表（抽屉）、会话管理 |
| **聊天 API** | 聊天界面、流式输出、工具审批弹窗 |
| **消息 API** | 历史记录加载、消息搜索 |
| **工作区 API** | 文件管理界面、上传/下载 |

## 🎨 设计规范

### 颜色使用
```kotlin
// 主色调
Primary                 // 主要按钮、强调元素
PrimaryContainer        // 用户消息背景
OnPrimaryContainer      // 用户消息文字

// 表面色
Surface                 // 卡片背景
SurfaceVariant          // AI 消息背景
OnSurface               // 主要文字
OnSurfaceVariant        // 次要文字

// 语义色
Error                   // 错误、删除
ErrorContainer          // 错误背景
Success (自定义)        // 成功状态
Warning (自定义)        // 警告状态
```

### 圆角规范
```kotlin
小圆角: 8.dp    // 小按钮、标签
中圆角: 12.dp   // 输入框、普通卡片
大圆角: 16.dp   // 主要按钮、大卡片
超大圆角: 24.dp  // Logo、特殊元素
```

### 间距规范
```kotlin
极小: 4.dp      // 文字间距
小: 8.dp        // 元素内边距
中: 12-16.dp    // 卡片内边距、列表间距
大: 24.dp       // 页面边距
超大: 32-48.dp  // 区块间距
```

### 阴影规范
```kotlin
无阴影: 0.dp           // 扁平卡片
轻微: 1-2.dp           // 列表项
中等: 4-6.dp           // 悬浮卡片
强烈: 8.dp             // 重要弹窗
```

### 动画时长
```kotlin
快速: 150ms     // 按钮点击
正常: 300ms     // 页面切换
慢速: 500ms     // 复杂动画
```
